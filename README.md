## 🌟 Overview

A **dark-themed, animated personal portfolio** built from scratch with pure HTML, CSS, and JavaScript — no frameworks, no dependencies. Designed to showcase my journey as an AI student and web developer, with smooth animations, interactive sections, and a fully responsive layout.

> *"Passionate about artificial intelligence and web development. Building innovative solutions from Nepal."*

| Section | Description |

| ⚡ **Skills** | Categorised skill bars with live-fill animations |

## ⚡ Features

- 🎨 **Dark glassmorphism UI** with cyan/purple gradient accents
- ✨ **Particle canvas** background with connected node animation
- 🌊 **Scroll reveal animations** on every section
- 📊 **Animated skill bars** that fill on scroll into view
- 🔄 **Project filter** — filter by Web / AI / All
- 📱 **Fully responsive** — mobile, tablet, desktop
- ⏱️ **Education timeline** with animated line fill
- 📋 **Click-to-copy** email functionality
- 🌐 **Floating orbit rings** on profile card
